# 一、docker镜像管理

#### 1.1、镜像搜索-search


从docker镜像仓库模糊搜索镜像

用法：
docker search 镜像关键字

```
docker search ubuntu
```

![](/assets/img/Pasted image 20251209224642.png)

**字段说明**
NAME：镜像名称
DESCRIPTION：镜像描述
STARS：镜像星级，数字越大代表用的人越多
OFFICIAL：是否是官方，有[OK]代表官方


#### 1.2、镜像下载-pull

从docker指定的仓库下载镜像到本地

用法：
docker pull 镜像名称

```
docker pull ubuntu
```

![](/assets/img/Pasted image 20251209225504.png)

#### 1.3、本地镜像查看-images

查看本地存储的镜像
```
docker images
```

![](/assets/img/Pasted image 20251209225614.png)

**字段说明**
REPOSITORY:镜像的名字
TAG：镜像的标签
IMAGE ID：镜像的ID号
CREATED：镜像建立时间
SIZE: 镜像大小

#### 1.4、镜像详细信息查看-inspect

用来显示镜像的详细信息

![](/assets/img/Pasted image 20251209230123.png)

#### 1.5、镜像保存-save

保存镜像为压缩文件

用法：
docker save -o 压缩文件名称 [镜像名称或ID]

```
docker save -o ubuntu_base.tar ubuntu:latest
```

![](/assets/img/Pasted image 20251209234951.png)

#### 1.6、本地镜像删除-rmi

删除本地某个镜像

用法：
docker rmi [镜像名称或者ID]

```
docker rmi ubuntu:latest
```

![](/assets/img/Pasted image 20251209235733.png)


#### 1.7、镜像导入-load

用法：
docker load -i 镜像压缩文件名称 [镜像名称或ID]

```
docker load -i ubuntu_base.tar
```

![](/assets/img/Pasted image 20251209235844.png)
