**tmux作用：**

在Linux的一个终端打开多个虚拟终端，这些终端即使断开ssh，也不会停止运行。代替screen。

普通终端关闭或者断开SSH连接以后，程序会被直接终止，而tmux的程序可以继续运行。


### 安装tmux

```
apt install -y tmux
```

### tmux基本使用

**1、启动并进入一个tmux会话**
```
tmux
```

或者给会话命名
```
tmux new -s [会话名]
```

**2、常用快捷键**

tmux的各种操作都要搭配前缀键`Ctrl + b`

**创建新窗口**

`Ctrl + b c`

**切换窗口(n上一个窗口，p下一个窗口)**

`Ctrl + b n / p`

也可以按编号切

`Ctrl + b 0~9`

**横向分屏**

`Ctrl + b "`

**纵向分屏**

`Ctrl + b %`

**切pane**

`Ctrl + b 方向键`

**临时离开（不中断会话）**

`Ctrl + b d`

**复制模式(查看历史输出)**

`Ctrl + b [`

复制模式的常用操作：

**上下滚动：** 上下方向键
**开始选择：** 空格键
**复制：** enter键
**退出：** q

**关闭面板**

`exit`

### 会话操作命令

**查看会话**
```
tmux ls
```

**进入会话**

```
tmux attach -t 会话名
```

**杀死会话**

```
tmux kill-session -t 会话名
```

如果要清除所有会话
```
tmux kill-server
```


**掌握这些操作和命令，tmux就已经熟练使用了**
