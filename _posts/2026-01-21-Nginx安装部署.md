**一、Nginx源码下载**

- Nginx官网 ：<https://nginx.org/>
- 源码包下载 ： https://nginx.org/download/nginx-1.28.1.tar.gz
- 操作系统 ： alma linux 10


**二、安装Nginx**

```
# 1、下载nginx源码包
wget https://nginx.org/download/nginx-1.28.1.tar.gz -P /usr/src && cd /usr/src

# 2、安装nginx依赖包
#  - gcc: 源码编译工具
#  - pcre-devel： nginx url_rewrite 功能提供包   
#  - zlib-devel： nginx 压缩功能提供包
    
# 以前centos安装依赖的命令 ：dnf install -y gcc pcre-devel zlib-devel

dnf install -y gcc pcre2-devel zlib-devel

# 3、解压nginx源码，并进入源码包
tar zxf nginx-1.28.1.tar.gz && cd nginx-1.28.1

# 4、配置nginx源码
# 目的：
#   1) 检查环境是否满足安装条件
#   2) 指定安装方式，包括nginx安装目录、配置文件以及其他各种文件位置、模块启用等，这里只定义了安装位置
./configure --prefix=/usr/local/nginx

# 5、编译nginx源码
make -j4

# 6、安装nginx
make install 
```

关于预编译时使用的参数文档：<https://nginx.org/en/docs/configure.html>



**三、Nginx相关目录**

  nginx path prefix: "/usr/local/nginx"
  
  nginx binary file: "/usr/local/nginx/sbin/nginx"
  
  nginx modules path: "/usr/local/nginx/modules"
  
  nginx configuration prefix: "/usr/local/nginx/conf"
  
  nginx configuration file: "/usr/local/nginx/conf/nginx.conf"
  
  nginx pid file: "/usr/local/nginx/logs/nginx.pid"
  
  nginx error log file: "/usr/local/nginx/logs/error.log"
  
  nginx http access log file: "/usr/local/nginx/logs/access.log"
  
  nginx http client request body temporary files: "client_body_temp"
  
  nginx http proxy temporary files: "proxy_temp"
  
  nginx http fastcgi temporary files: "fastcgi_temp"
  
  nginx http uwsgi temporary files: "uwsgi_temp"
  
  nginx http scgi temporary files: "scgi_temp"


```
killall -s QUIT nginx
```



**五、nginx启动测试**
