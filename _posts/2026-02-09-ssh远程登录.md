linux一般作为服务器使用，服务器一般放在机房，你不可能在机房操作Linux服务器，这种情况就需要远程登录到Linux服务器来管理系统。

Linux系统通过SSH服务实现远程登录的功能，默认SSH服务的端口号为`22`。

windows系统可以用来远程登录Linux的客户端有putty、bitvise、Termius等。


ssh登录前提：前提：目标安装并启动ssh服务


在终端使用ssh远程登录服务器

```
# ssh -p ssh端口号 主机用户名@主机IP
ssh -p 22 root@127.0.0.1
```

输入用户名密码即可登录



密钥登录(推荐)

1、打开终端，生成密钥对
```
ssh-keygen -t rsa -b 4096 -C "email@example.com"
```

-t 加密类型，-b 密钥长度，越大越安全，-C 注释信息


系统提示：
```
Enter file in which to save the key (/home/user/.ssh/id_rsa):
```

直接回车，使用默认路径即可。


系统提示：
```
Enter passphrase (empty for no passphrase):
```

不想设置密码直接回车留空，但是真实情况下不推荐


生成后在`~/.ssh/`目录下得到两文件：
- 私钥：`id_rsa`，绝对不能泄露
- 公钥：`id_rsa.pub`，用来上传到远程服务器



2、上传公钥到远程服务器

```
ssh-copy-id -i ~/.ssh/id_rsa.pub 用户名@远程IP
```

系统提示输入远程用户密码，完成后公钥会自动追加到远程服务器的 `~/.ssh/authorized_keys` 文件。

3、测试免密登录

在本地执行：
```
ssh 用户名@远程IP
```

如果配置成功，将直接登录远程服务器，不需要输入密码。但是如果之前设置了私钥passphrase，需要输入。


如果有多个密钥
```
ssh -i ~/.ssh/id_rsa_another 用户名@远程IP
```
