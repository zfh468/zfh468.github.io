**一、文件内容查看**

1.1、cat

从第一行开始显示文件内容

```
# 在终端显示file的内容
cat file
# 同时显示file1和file2的内容
cat file1 file2
# 将file1和file2的内容合并放入文件file
cat file1 file2 > file
```

1.2、tac

从最后一行开始显文件内容，用法与cat类似
```
tac file
```


1.3、nl

从第一行开始显示文件内容，但会给每行内容前加上行号。

```
nl file
```


1.4、more

more是一个分屏阅读器，可以一页一页的查看内容
```
more file
```

常用操作：
- 空格键space，向下翻一页
- enter回车键，向下滚动一行
- b键，向上翻一页
- q键，退出阅读
- /关键词，在当前文件里搜索内容，按n继续寻找下一个
- =，显示当前行号
- v键，直接调用vi编辑器修改

less比more好用，more对向上翻页的支持不是很好



1.5、less

less和more的作用类似，但功能更多，占用系统资源更少。相比more，less最强大的地方在于它是按需加载的，不需要加载整个文件，即使打开一个10GB的日志文件也能秒开，而且支持随意上下翻页和双向匹配。

直接打开
```
less file
```

查看多个文件，输入:n切换下一个，:p切换上一个
```
less file1 file2 
```

慢慢查看执行过的历史命令
```
history | less
```

交互键：
- j/k，逐行下移/上移，也可以上下方向键
- space / b，整页下翻 / 上翻
- g / G，直达文件开头 / 结尾
- /关键词，向下搜索
- ?关键词，向上搜索
- n / N，跳转到下一个/ 上一个匹配项
- F，实时滚动模式，类似tail -f

常用参数：
- -N，显示行号
- -i忽略大小写



1.6、head

查看文件前几行内容，默认显示前10行

```
# 查看file前20行内容
head -n 20 file
```


1.7、tail

查看文件后面几行内容，默认显示最后10行

```
# 查看file最后20行内容
tail -n 20 file

# 显示file的最后10个字符
tail -c 10 file

# 实时监控
tail -f file
```
