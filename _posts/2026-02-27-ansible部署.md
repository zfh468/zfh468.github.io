**ansible介绍**

ansible是一个基于python开发的自动化运维工具，由Red Hat维护，主要用于批量服务器配置管理/应用部署/自动化运维和云资源管理。它不需要客户端安装agent（使用ssh），用yaml编写playbook，简单易读。


**一、准备**
一台Ubuntu24.04
三台alma linux 9

**二、Ansible部署**

2.1、Ubuntu安装ansible
```
sudo apt update
sudo apt install -y ansible
```

验证安装
```
ansible --version
```


Inventory编写

Inventory是定义被管理节点主机的文件，默认位置在/etc/ansible/hosts，不过我选择在当前目录写一个hosts
```
# 分组写法，方便分类批量管理
# 除了ip也可以写域名
[web]
192.168.1.221 
192.168.1.222 
192.168.1.223 
```

2.2、配置ssh免密登录
```
# 一路回车即可
ssh-keygen -t ed25519
```

把密钥发送给所有被管理节点
```
ssh-copy-id root@192.168.1.221
ssh-copy-id root@192.168.1.222
ssh-copy-id root@192.168.1.223
```

测试
```
ansible all -i hosts -m ping 
```
