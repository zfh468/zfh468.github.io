
### 1、tcpdump是什么

tcpdump是一个命令行抓包工具，用于捕获网络接口上的数据包并显示出来。


常用于：
- 分析网络问题
- 网络安全数据包分析
- 学习网络协议
- 查看服务器是否收到请求


**tcpdump**是一个网络监听器，可以看到网卡上经过的所有数据包。



### 2、tcpdump常用命令


抓包

-i指定网卡，第一个-n不解析域名，第二个-n不把端口转换成服务名称（不把80转换成http）。提高效率，减少干扰

```
tcpdump -i eth0 -nn
```


抓100个包
```
tcpdump -i eth0 -nn -c 100
```


抓80端口
```
tcpdump -i eth0 port 80 -nn
```


保存抓包
```
tcpdump -i eth0 -w test.pcap
```


读取抓包文件
```
tcpdump -r capture.pcap
```


抓ip
```
tcpdump host 192.168.1.10
```


抓端口
```
tcpdump port 80
```


抓协议，tcp协议
```
tcpdump tcp
```


显示更详细数据
```
tcpdump -v
```

想更详细`tcpdump -vv`



查看数据内容，会显示ASCII、十六进制
```
tcpdump -x
```



### 3、组合命令


抓某ip的80端口
```
sudo tcpdump host 192.168.1.10 and port 80
```


抓80或443
```
tcpdump port 80 or port 443
```


排除ip
```
tcpdump not host 192.168.1.10
```
