
#### 1、vnc介绍

VNC (Virtual Network Console)是虚拟网络控制台的缩写。它 是一款优秀的远程控制工具软件，由著名的 AT&T 的欧洲研究实验室开发的。VNC 是在基于 UNIX和 Linux操作系统的免费的开源软件，远程控制能力强大，高效实用，其性能可以和 Windows和 MAC中的任何远程控制软件媲美


#### 2、vnc连接

宿主机放行5980
```
# 1. 永久放行 5980 端口
firewall-cmd --permanent --add-port=5980/tcp

# 2. 重新加载防火墙配置使生效
firewall-cmd --reload

# 3. 确认端口是否已处于监听状态
netstat -tpln | grep 5980
```


确认vm1正常启动，以及vnc配置是否生效
```
# 1. 查看虚拟机列表及状态
virsh list --all

# 2. 如果虚拟机没启动，手动启动它
virsh start vm1

# 3. 检查虚拟机的显示配置
virsh vncdisplay vm1
```

vncdisplay显示`：80`，代表端口号是5900+80=5980
