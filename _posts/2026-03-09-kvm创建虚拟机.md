virt-install 命令安装
```
dnf install -y virt-install
```

#### 1、virt-install创建虚拟机

使用镜像创建虚拟机

```
virt-install \
--name vm1 \
--memory 2048 \
--vcpus 2 \
--disk path=/var/lib/libvirt/images/vm1.qcow2,size=10,format=qcow2,bus=virtio \
--os-variant rhel8.0 \
--cdrom /cache/iso/CentOS-8-x86_64-1905-dvd1.iso \
--network network=default,model=virtio,mac=52:54:00:A7:9C:01 \
--graphics vnc,listen=0.0.0.0,port=5980,password=123456 \
--noautoconsole \
--autostart
```


字段说明：
```
--name   虚拟机名称
--memory 分配的内存大小，默认单位MB
--vcpus  虚拟机cpu数量，不要超过物理机核数，默认是1
--disk   
	path 指定磁盘存放位置，建议指定，方便管理
	size=10   10GB大小
	format=qcow2   qcow2相比raw格式，支持快照和写时复制，且文件大小会随实际使用量增长
	bus=virtio    启用半虚拟化驱动，可以显著提升性能
--os-variant rhel8.0     告诉hypervisor目标系统内核特性，从而自动配置最佳硬件环境	
--cdrom  指定系统安装镜像文件位置
--network
	network=default   使用libvirt默认的NAT网络，通常网段是192.168.122.x
	model=virtio   优化网络性能，让网卡跑在virtio模式下，降低CPU消耗并提升吞吐
	mac   手动指定mac地址
--graphics vnc
	listen=0.0.0.0  允许任何IP访问vnc桌面，只要放行5980端口
	port=5980  明确端口号，避免kvm自动分配找不到入口
--noautoconsole 运行命令后立即回到终端提示符
--autostart  虚拟机虽宿主机开机自启
```



#### 2、kvm相关目录及虚拟机配置文件

**2.1、相关目录**

kvm虚拟机目录：`/var/lib/libvirt/`

虚拟机配置文件（XML）目录：`/etc/libvirt/qemu/`

运行日志目录（报错排查常用）：`/var/log/libvirt/qemu/`

镜像及磁盘目录：`/var/lib/libvirt/images/`

libvirt目录：`/etc/libvirt`

![](/assets/img/Pasted image 20260309171600.png)

- **`/etc/libvirt/libvirtd.conf`**：Libvirt 守护进程的主配置文件，控制远程访问和安全校验。
    
- **`/etc/libvirt/qemu.conf`**：QEMU 驱动的特定配置，例如 VNC 的监听地址设置。
    
- **`/var/run/libvirt/qemu/`**：存放正在运行的虚拟机的临时状态文件（如 PID 文件和 Socket 文件）。
