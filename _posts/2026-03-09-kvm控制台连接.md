### 1、介绍

在 KVM 中，**控制台连接（Serial Console）** 是比 VNC 更底层的管理方式。它不依赖图形界面，直接通过字符设备与虚拟机的内核通信。


### 2、操作

通过vnc连接vm1,执行
```
systemctl start serial-getty@ttyS0.service
systemctl enable serial-getty@ttyS0.service
```


在宿主机执行
```
virsh console vm1
```

进去后多次回车，输入vm1的系统账号密码即可登录
