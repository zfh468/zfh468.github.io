在把虚拟机vm1作为模板进行克隆前，需要对里面的一些文件进行清理，避免冲突


安装好一台干净的系统，执行 `virt-sysprep` 清理掉敏感信息（如旧的网卡配置、主机名、日志等），否则

```
dnf install -y guestfs-tools
```


清理模板机
```
# 确保虚拟机已关机 
virsh shutdown vm1
# 如果shutdown不管用再使用destroy
virsh destroy vm1

# 清理残余信息（如 SSH 密钥、旧网络配置）
virt-sysprep -d vm1
```

执行完以上步骤，vm1现在可以作为模板机进行克隆



**模板机创建虚拟机**

1、创建模板磁盘副本
```
cp /var/lib/libvirt/images/vm1.qcow2 /var/lib/libvirt/images/vm2.qcow2
```

2、创建虚拟机
```
virt-install \
--name vm2 \
--memory 2048 \
--vcpus 2 \
--disk path=/var/lib/libvirt/images/vm2.qcow2,format=qcow2 \
--import \
--os-variant rhel8.0 \
--network network=default
```

--import表示从已有磁盘启动，vm2.qcow2是从模板复制来的


检查
```
virsh list --all
```
