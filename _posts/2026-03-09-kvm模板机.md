在把虚拟机vm1作为模板进行克隆前，需要对里面的一些文件进行清理，避免冲突


安装好一台干净的系统，执行 `virt-sysprep` 清理掉敏感信息（如旧的网卡配置、主机名、日志等），否则

```
dnf install -y guestfs-tools
```


清理模板机
```
# 确保虚拟机已关机 
virsh shutdown vm1
# 如果shutdown不管用再使用destroy
virsh destroy vm1

# 清理残余信息（如 SSH 密钥、旧网络配置）
virt-sysprep -d vm1
```

执行完以上步骤，vm1现在可以作为模板机进行克隆
