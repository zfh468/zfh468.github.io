#### 一、虚拟机管理工具

工具：
- virt-manager
- cockpit web控制台
- virsh命令



#### 二、virsh常用命令

**2.1 基本管理命令**

查看所有虚拟机
```
virsh list --all
```


虚拟机启动、关闭、重启
```
# 启动虚拟机
virsh start 虚拟机名

# 关闭虚拟机
virsh shutdown 虚拟机名

# 强制关闭虚拟机
virsh destroy 虚拟机名

# 重启虚拟机，前提必须是启动状态
virsh reboot 虚拟机名
```


挂起/恢复
```
# 暂停虚拟机运行或从暂停中恢复
virsh suspend/resume 虚拟机名
```


自启动
```
# 随宿主机启动而自动启动
virsh autostart 虚拟机名
```



**2.2 配置与修改**

kvm虚拟机的本质是一个xml配置文件和一个磁盘文件



编辑虚拟机配置
```
# 修改内存大小、CPU 核心数、网卡模式、VNC 端口等。修改后通常需要重启虚拟机生效。
virsh edit 虚拟机名
```


导出配置(备份)
```
virsh dumpxml 虚拟机名 > backup.xml
```


根据xml配置文件创建虚拟机
```
virsh define backup.xml
```


删除虚拟机（只删除配置文件，不会删除qcow2磁盘文件，需要手动删）
```
virsh undefine 虚拟机名
```





**2.3 连接与交互**

控制台连接
```
# 直接通过串口连接，退出快捷键Ctrl + ]
vish console 虚拟机名
```


查看虚拟机vnc连接端口
```
virsh vncdisplay 虚拟机名
```
