### 二、桥接网络

为虚拟机启用物理网卡网络


2.1、在宿主机创建网桥
```
# 1、创建网桥接口br0
nmcli connection add type bridge con-name br0 ifname br0

# 2、将物理网卡绑定到网桥
# 假设你的物理网卡名为 ens224
执行此操作时，物理机的网络会断开一下，建议在物理机本地操作。
nmcli connection add type bridge-slave con-name br0-slave ifname ens224 master br0

# 查看ens224的连接名，为 Wired connection 1
nmcli connection show

# 3、启用网桥并关闭物理网卡原配置
nmcli connection up br0
nmcli connection down ens224连接设备名或者UUID  # 以后只使用 br0 进行通信


# 此时宿主机IP会转移到br0上，可以使用ip a命令查看
```
