
### kvm快照管理

kvm自带快照功能

进行危险操作前，建议先做快照


虚拟机创建快照
```
virsh snapshot-create-as --domain 虚拟机名 --name 快照名 --description "说明信息"
```



查看虚拟机快照
```
virsh snapshot-list <VM名字>
```


恢复快照
```
virsh snapshot-revert <VM名字> <快照名>
```


删除快照
```
virsh snapshot-delete <VM名字> <快照名>
```



### 磁盘格式转换

查看虚拟机磁盘信息
```
qemu-img info 磁盘文件
```


转换磁盘格式最核心的工具是 `qemu-img`


格式转换命令语法
```
qemu-img convert -f [原格式] -O [目标格式] [原文件路径] [新文件路径]
```





raw转换成qcow2
```
qemu-img convert -f raw -O qcow2 source_disk.raw target_disk.qcow2
```


raw比qcow2性能好，但不支持快照，且直接占用全部物理空间



转换完后使用qemu-img info命令检查磁盘


**磁盘格式转换以后，原来的虚拟机不知道，需要更新xml**

```
virsh edit <虚拟机名字>
```


找到driver这一行，把type=raw改成qcow2
```
<driver name='qemu' type='qcow2'/>
<source file='/var/lib/libvirt/images/target_disk.qcow2'/>
```

保存退出并重新启动虚拟机就好了
